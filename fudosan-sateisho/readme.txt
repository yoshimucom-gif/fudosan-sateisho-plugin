=== 不動産 査定書作成受付 ===

査定書の作成を受け付けるフォーム。物件情報とメールアドレスを受け取り、
お客様へ「受付完了メール」を自動返信し、担当者へ通知します。
査定書は後日スタッフが作成して、お客様のメール宛に送付します。
（AIによる価格の即時表示は行いません＝匿名査定プラグインとは別物です）

※ 匿名の「かんたん不動産AI査定」プラグインとは別プラグインです。
   両方を同じサイトで併用できます（メニュー・受付データは別々）。


■ 導入手順
1. 「プラグイン」→「新規追加」→「プラグインのアップロード」→ この zip を
   選択してインストール →「有効化」
2. 左メニュー「査定書作成受付 → 設定」で以下を設定
   - サイト名／運営者名／問い合わせ先
   - 送信元メール（お客様への受付完了メールの差出人）
   - 通知先メール（担当者。受付が届いたら通知）
   - プライバシーポリシーURL・利用規約URL
3. 受付フォームを出したい固定ページに、ショートコードを貼る

       [fudosan_sateisho]


■ ショートコード（デザインパターン）
   [fudosan_sateisho]                  標準。全項目・幅100%・枠なし
   [fudosan_sateisho design="compact"] 必須＋築年のみ・幅440pxのカード
   [fudosan_sateisho design="card"]    全項目を枠＋影のカードで表示
   属性: button="ボタン文言"


■ 設定タブ
   基本設定           : サイト名・運営者・送信元/通知先メール・ポリシーURL
   表示項目・対象エリア : 任意項目の表示ON/OFF、対象都道府県の限定（空なら全国）
   自動返信メール      : お客様向け「受付完了メール」の件名・本文／テスト送信
   デザイン           : ボタン色・見出し色・必須バッジ色
   使い方             : 本ファイルと同じ内容

■ 受付データ
   左メニュー「査定書作成受付 → 受付一覧」
   CSVエクスポート（Excel向け）と、行ごとの削除（個人情報の削除依頼対応）に対応。
   データは自サイトのDB（wp_fudosan_sateisho_leads）にのみ保存され、外部送信されません。

■ 受付後の流れ
   送信 → お客様へ受付完了メールを自動返信 ＋ 通知先メールへ受付内容を通知
        → 担当者が査定書を作成し、後日お客様へ送付。

■ メール到達率について
   サーバーから直接送るメールは迷惑メール判定されやすいため、
   「WP Mail SMTP」等で SMTP 送信にし、送信ドメインの
   SPF / DKIM / DMARC を設定してください。
   設定→自動返信メールタブの「テストメールを自分宛に送信」で確認できます。

■ 通信方式
   フォーム送信は admin-ajax.php 経由です（REST APIを無効化している
   セキュリティ設定〔SiteGuard等〕の環境でも動作します）。

■ 法的な注意
   作成する査定書は宅建業の「価格査定（参考価格）」であり、
   不動産鑑定士による「鑑定評価」ではありません。免責文は削除しないでください。
   公開前に弁護士等の確認を推奨します。
